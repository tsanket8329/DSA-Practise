Problem (Non-Divisible Subset – HackerRank)

Given:

Array s
Integer k

👉 Find the maximum subset size such that:

(a + b) % k ≠ 0   for any pair
💡 Final Approach (Short)
Count frequency of remainders: freq[i] = count of numbers with remainder i
Add:
1 if freq[0] > 0
For i = 1 → k/2:
If i == k - i → add 1
Else → add max(freq[i], freq[k - i])
🚀 FINAL JAVA CODE
public static int nonDivisibleSubset(int k, List<Integer> s) {
    int[] freq = new int[k];

    // Count remainders
    for (int num : s) {
        freq[num % k]++;
    }

    int result = 0;

    // Handle remainder 0
    if (freq[0] > 0) {
        result += 1;
    }

    // Handle pairs
    for (int i = 1; i <= k / 2; i++) {
        if (i == k - i) {
            if (freq[i] > 0) result += 1;
        } else {
            result += Math.max(freq[i], freq[k - i]);
        }
    }

    return result;
}
📌 Sample
Input:
4 3
1 7 2 4
Output:
3
🔥 Final One-Line Concept

Use remainders and never pick both r and k-r.
